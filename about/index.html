About the author

Hi, I'm HiPhish and this is my website. I'm a mathematician and a programmer and my motto is I want to to exist, so I'll make it. My goal is rock-solid software that is reliable and fit for productive use, because I wouldn't want to use anything else myself.

I chose the Workshop motto because I approach programming as a craft. This site is where I exhibit my work, present my services, and if you follow the links to my repositories you can see the details or how it is made. Programmimg is a new form of craftmanship and the computer is our workbench.

Tools of the trade

My code editor of choice is Neovim, which I use for most text editing as well. For version control I use Git and my package manager is Homebrew. This site was generate using the Pelican static site generator. My terminal is xterm running in iTerm 2. Other than these general tools I use what is appropriate for the given task.

About this website

The Workshop has been generated using the Pelican static site generator with a custom theme. The theme is still work in progress and I cannot complete it before I don't write a plugin for Pelican. Once the theme has all the templates needed to be complete I will release it to the public.

A big help during development was Twitter Bootstrap, a CSS framework for creating responsive web pages. This site uses the modified style sheet flatly by Bootswatch.

The "holy grail" blog layout and the sticky footer were achieved using code examples from Philip Walton's Solved by Flexbox.

The gallery uses Stephane Caron's PrettyPhoto Javascript library.

What is missing

I want the Workshop to be a central home for all my projects, commercial and free. This would also require maintaining blogs for news and updates, these blogs can then generate RSS feeds for people to subscribe to.

The problem is that Pelican only allows the creation of one blog. People would constantly get spammed by updates about products they don't care about and that would be simply unacceptable. To achieve multiple blogs I will have to write a Pelican plugin.

This will take a while though, I have to learn enough Python and familiarise myself with the Pelican ecosystem. There is no point in holding back the site just because of that, so I'm releasing it for now the way it is.

LibreJS - Free JavaScript

The Workshop is build from free (libre) software that respects the user's freedom. The same also applies to the JavaScript employed on this site. I have made sure that the site complies with the LibreJS specifications. You can find a list of scripts employed under about/javascript.
